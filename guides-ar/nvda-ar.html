دليل البدء السريع لقارئ الشاشة NVDA

تم التحديث ليعكس ممارسات التنقل الحديثة واختبار الويب، استنادًا إلى وثائق NV Access الرسمية وإرشادات Deque.

تشغيل وإيقاف NVDA

  • تشغيل NVDA: Ctrl + Alt + N (افتراضيًا إذا تم التثبيت) أو من قائمة ابدأ.
  • إيقاف NVDA: مفتاح NVDA + Q ثم Enter.

التنقل الأساسي

  • إيقاف التحدث: زر Control
  • قراءة مستمرة من هنا: مفتاح NVDA + سهم للأسفل
  • الانتقال إلى العنصر التالي: سهم للأسفل
  • الرجوع إلى العنصر السابق: سهم للأعلى
  • تنشيط رابط/زر: Enter أو Spacebar

التنقل حسب عناصر الصفحة

  • العنوان التالي: H
  • العنوان التالي من المستوى 1–6: اضغط على الأرقام 1–6
  • الرابط التالي: K
  • الزر التالي: B
  • حقل النموذج التالي: F
  • القائمة التالية: L
  • المنطقة أو المعلم التالي: D

العمل مع النماذج

  • الدخول إلى وضع النماذج: يتم تلقائيًا عند دخول حقول النماذج
  • التبديل اليدوي بين وضع النماذج وتصفح: مفتاح NVDA + Spacebar
  • الانتقال إلى خانة اختيار: X
  • تحديد/إلغاء تحديد خانة: Spacebar
  • الانتقال إلى مربع اختيار: C
  • فتح مربع الاختيار: Alt + سهم للأسفل

التنقل داخل الجداول

  • الجدول التالي: T
  • التحرك يمين/يسار: Ctrl + Alt + سهم يمين/يسار
  • التحرك للأسفل/للأعلى: Ctrl + Alt + سهم للأسفل/للأعلى
  • قراءة الخلية الحالية: مفتاح NVDA + C

أوامر مفيدة أخرى

  • عرض جميع العناصر (روابط، عناوين، نماذج): مفتاح NVDA + F7
  • تبديل وضع التصفح/التركيز: مفتاح NVDA + Spacebar
  • تبديل وضع النطق (كتم مؤقت): مفتاح NVDA + S

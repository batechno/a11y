دليل البدء السريع لـ VoiceOver على macOS

تم التحديث لإصدارات macOS Ventura وSonoma (2023–2024). يركز على التنقل عبر الويب واحتياجات الاختبار اليومية.

تشغيل وإيقاف VoiceOver

  • تشغيل VoiceOver: Command + F5 (أو الضغط ثلاث مرات على زر Touch ID في أجهزة ماك الحديثة)
  • إيقاف VoiceOver: نفس الأمر: Command + F5

أساسيات VoiceOver

  • مفتاح VoiceOver: Control + Option (مفتاح VO)
  • إيقاف التحدث: Control
  • قراءة الكل من الموضع الحالي: VO + A

التنقل الأساسي

  • الانتقال إلى العنصر التالي: VO + سهم يمين
  • الرجوع إلى العنصر السابق: VO + سهم يسار
  • تنشيط العنصر المحدد: VO + مفتاح المسافة

اختصارات التنقل في الويب

  • العنوان التالي: VO + Command + H
  • الرابط التالي: VO + Command + L
  • عنصر التحكم في النماذج التالي: VO + Command + J
  • الجدول التالي: VO + Command + T
  • المنطقة أو المعلم التالي: VO + Command + R
  • الزر التالي: VO + Command + B

الروتور (قائمة التنقل السريعة)

  • فتح الروتور: VO + U
  • استخدم الروتور للانتقال السريع بين العناوين، الروابط، الحقول، المعالم، وغيرها.

التفاعل مع النماذج

  • بدء التفاعل مع حقل: VO + Shift + سهم للأسفل
  • إيقاف التفاعل: VO + Shift + سهم للأعلى

التنقل داخل الجداول

  • التحرك بين الخلايا: بعد بدء التفاعل، استخدم VO + الأسهم
  • قراءة الخلية الحالية: VO + C

أوامر مفيدة أخرى

  • قراءة العنصر الحالي: VO + F3
  • فتح قائمة المساعدة: VO + H
  • كتم أو إلغاء كتم الصوت: VO + Shift + S

دليل البدء السريع لقارئ الشاشة الراوي (Narrator)

تم التحديث لنظام التشغيل Windows 11 (2023–2024). يركز على الاستخدام الأساسي لقارئ الشاشة لمستخدمي Windows.

تشغيل وإيقاف الراوي

  • تشغيل الراوي: Ctrl + مفتاح شعار Windows + Enter
  • إيقاف الراوي: نفس المفاتيح أو من إعدادات الراوي.

التنقل الأساسي

  • قراءة كل شيء من الموقع الحالي: مفتاح الراوي + سهم للأسفل
  • إيقاف التحدث: Control
  • الانتقال إلى العنصر التالي: Tab
  • الرجوع إلى العنصر السابق: Shift + Tab
  • تنشيط رابط أو زر: Enter أو Spacebar

التنقل حسب العناصر

  • العنوان التالي: H
  • الرابط التالي: K
  • الزر التالي: B
  • حقل النموذج التالي: F
  • الجدول التالي: T

وضع النماذج (وضع المسح الضوئي)

  • تبديل وضع المسح: مفتاح الراوي + Spacebar
  • أثناء تفعيل وضع المسح: استخدم مفاتيح الأسهم للتنقل وتنشيط العناصر بسهولة.

أوامر مفيدة أخرى

  • عرض الروابط والعناوين والمعالم: مفتاح الراوي + F6
  • فتح إعدادات الراوي: مفتاح الراوي + Ctrl + N
  • تغيير سرعة الصوت: مفتاح الراوي + زائد (+) أو ناقص (–)

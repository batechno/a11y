ZoomText Quick Start Guide

Updated for ZoomText 2024 (Fusion 2024 Suite). Focused on magnification, enhancement, and basic speech features.

Starting and Stopping ZoomText

  • Start ZoomText: Launch from Desktop or Start Menu.
  • Pause Magnifier/Reader: Alt + Pause
  • Exit ZoomText: Alt + F4 from ZoomText toolbar, or File > Exit.

Magnification Controls

  • Increase Magnification: Alt + Up Arrow
  • Decrease Magnification: Alt + Down Arrow
  • Zoom to 1x (Normal View): Alt + Left Arrow

Magnifier Views

  • Full Screen Magnifier: Magnifies entire screen.
  • Lens View: Magnifies a rectangular area under your pointer.
  • Docked View: Magnifies the top, bottom, or a portion of the screen.
  • Toggle View Modes: Alt + Z

Color and Contrast Enhancements

  • Apply Color Filters: Use ZoomText > Color settings.
  • Invert Colors: Quick toggle available under Color settings.
  • Reduce Glare: Adjust brightness/contrast settings as needed.

Navigation and Focus Enhancements

Narrator Quick Start Guide

Updated for Windows 11 (2023–2024). Focused on basic screen reader usage for Windows users.

Turning Narrator On/Off

  • Start Narrator: Ctrl + Windows Logo Key + Enter
  • Turn Off Narrator: Press the same keys again or Exit from Narrator settings.

Basic Navigation

  • Read All (from current location): Narrator Key + Down Arrow
  • Stop Speaking: Control
  • Move to Next Item: Tab
  • Move to Previous Item: Shift + Tab
  • Activate Link or Button: Enter or Spacebar

Navigating by Elements

  • Next Heading: H
  • Next Link: K
  • Next Button: B
  • Next Form Field: F
  • Next Table: T

Forms Mode (Scan Mode)

  • Toggle Scan Mode On/Off: Narrator Key + Spacebar
  • While Scan Mode is On: Use arrow keys to move and activate elements easily.

Other Useful Commands

  • List Links, Headings, Landmarks: Narrator Key + F6
  • Open Narrator Settings: Narrator Key + Ctrl + N
  • Change Voice Speed: Narrator Key + Plus (+) or Minus (–)

JAWS Screen Reader Quick Start Guide

Updated for modern web navigation and testing practices. Based on Deque and Freedom Scientific guidance.

Starting and Stopping JAWS

  • Start JAWS: Launch from desktop or Start menu (or use hotkey if set).
  • Turn off JAWS: Insert + F4, then Enter.

Basic Navigation

  • Stop speaking: Control
  • Read continuously from here: Insert + Down Arrow
  • Move to next item: Down Arrow
  • Move to previous item: Up Arrow
  • Activate link or button: Enter or Spacebar

Navigating by Page Elements

  • Next Heading: H
  • Next Heading of Level 1–6: Press 1–6
  • Next Link: Tab (or Insert + F7 to list all links)
  • Next Landmark/Region: R
  • Next Table: T
  • Next List: L
  • Next List Item: I
  • Next Button: B

Working with Forms

  • Enter Forms Mode: Press Enter when on a form field
  • Exit Forms Mode: Numpad Plus (+)
  • Next Form Field: F
  • Next Checkbox: X
  • Select/Deselect Checkbox: Spacebar
  • Next Combo Box: C
  • Open Combo Box: Alt + Down Arrow

Table Navigation

  • Next Table: T
  • Move Right/Left: Ctrl + Alt + Right/Left Arrow
  • Move Down/Up: Ctrl + Alt + Down/Up Arrow
  • Say Current Cell: Ctrl + Alt + Numpad 5

Other Useful Commands

  • List All Links, Headings, Buttons, Graphics: Insert + F3
  • List All Links (only): Insert + F7
  • Toggle Virtual Cursor (Web vs. App mode): Insert + Z

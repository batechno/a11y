NVDA Screen Reader Quick Start Guide

Updated for modern web navigation and testing practices. Based on NV Access official documentation and Deque guidelines.

Starting and Stopping NVDA

  • Start NVDA: Ctrl + Alt + N (default if installed) or from Start Menu.
  • Exit NVDA: NVDA key + Q, then Enter.

Basic Navigation

  • Stop speech: Control
  • Read continuously from here: NVDA key + Down Arrow
  • Move to next item: Down Arrow
  • Move to previous item: Up Arrow
  • Activate link/button: Enter or Spacebar

Navigating by Page Elements

  • Next Heading: H
  • Next Heading of Level 1–6: Press 1–6
  • Next Link: K
  • Next Button: B
  • Next Form Field: F
  • Next List: L
  • Next Landmark/Region: D

Working with Forms

  • Enter Forms/Focus Mode: Automatic when entering form fields
  • Manual Focus Mode: NVDA key + Spacebar (toggle)
  • Next Checkbox: X
  • Select/Deselect Checkbox: Spacebar
  • Next Combo Box: C
  • Open Combo Box: Alt + Down Arrow

Table Navigation

  • Next Table: T
  • Move Right/Left: Ctrl + Alt + Right/Left Arrow
  • Move Down/Up: Ctrl + Alt + Down/Up Arrow
  • Say Current Cell: NVDA key + C

Other Useful Commands

  • List All Elements (Links, Headings, Form Fields): NVDA key + F7
  • Toggle Browse/Focus Mode: NVDA key + Spacebar
  • Toggle Speech Mode (mute NVDA quickly): NVDA key + S

Full List of Commands: NVDA Official User Guide

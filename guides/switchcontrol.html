Switch Control Quick Start Guide

Updated for iOS 17 and macOS Sonoma (2024). Designed for users needing alternative input methods with switches.

Turning Switch Control On/Off

  • On iPhone/iPad: Settings > Accessibility > Switch Control > Turn On.
  • On Mac: System Settings > Accessibility > Switch Control > Turn On.
  • Quick Toggle (iOS): Triple-click Side or Home button (if configured).

Setting Up Switches

  • Add Switch: Settings > Accessibility > Switch Control > Switches > Add New Switch.
  • Switch Source: External (Bluetooth switch, USB device) or Screen tap.
  • Assign Actions: Choose actions like Select, Move to Next Item, Back, etc.

Basic Navigation

  • Item Scanning: Items are highlighted automatically.
  • Select Highlighted Item: Activate primary switch when the desired item is highlighted.
  • Move to Next Item: Press switch if manually configured for Next Item.

Point Scanning (Advanced)

  • Horizontal and Vertical Crosshairs: Tap switch to lock position and select coordinates.
  • Useful for: Touching anywhere on the screen that's not part of a UI element.

Customizing Switch Control

  • Auto Scanning Speed: Adjust timing under Switch Control settings.
  • Group Items: Enable/Disable group item scanning for efficiency.
  • Customize Menu: Add actions like Scroll, Gestures, Home, Back, App Switcher, etc.

Other Useful Tips

  • Recipes: Create custom gesture sequences triggered by switch presses.
  • Switch Control Practice: Settings > Accessibility > Switch Control > Practice to learn safely without actions triggering.

Starting and Stopping VoiceOver

  • Start VoiceOver: Command + F5 (or Touch ID button triple-press on newer Macs)
  • Turn Off VoiceOver: Command + F5 again

VoiceOver Basics

  • VoiceOver Modifier Key: Control + Option (VO Key)
  • Stop Speaking: Control
  • Read All From Current Position: VO + A

Basic Navigation

  • Move to Next Item: VO + Right Arrow
  • Move to Previous Item: VO + Left Arrow
  • Activate Selected Item: VO + Spacebar

Web Navigation Shortcuts

  • Next Heading: VO + Command + H
  • Next Link: VO + Command + L
  • Next Form Control: VO + Command + J
  • Next Table: VO + Command + T
  • Next Landmark/Region: VO + Command + R
  • Next Button: VO + Command + B

Rotor (Quick Navigation Menu)

  • Open Rotor: VO + U
  • Use Rotor to quickly jump by Headings, Links, Form Fields, Landmarks, etc.

Forms Interaction

  • Interact with Form Field: VO + Shift + Down Arrow
  • Stop Interacting: VO + Shift + Up Arrow

Table Navigation

  • Move Across Table Cells: Interact first (VO + Shift + Down Arrow), then VO + Arrow Keys
  • Read Current Cell: VO + C

Other Useful Commands

  • Read Current Item: VO + F3
  • Open VoiceOver Help Menu: VO + H
  • Mute/Unmute Speech: VO + Shift + S
